# ES

4ストロークエンジンの始動〜アイドル安定化までの過渡挙動を、Rustでリアルタイムに確認するためのターミナルシミュレータです。

## できること

- スターターモーターによるクランキング
- 初爆条件（スターターON + 点火ON + 燃料ON + 一定回転/吸気圧）での着火
- 着火後のトルク生成と摩擦トルクのつり合いによる回転上昇/安定化
- 吸気系（スロットル + インマニ）とエンジン吸入の連成
- 排気圧を境界条件にした簡易排気系
- VVT（吸気/排気）による充填効率変化
- p-V 線図のリアルタイム表示（近似）
- RPMトレンドのリアルタイム表示

## 実行

```bash
cargo run
```

## 操作キー

- `q`: 終了
- `s`: スターター ON/OFF
- `i`: 点火 ON/OFF
- `f`: 燃料 ON/OFF
- `w` / `x`: スロットル開度 増/減
- `a` / `z`: 吸気VVT 進角/遅角
- `k` / `m`: 排気VVT 進角/遅角

## モデル概要

- **吸気**: オリフィス近似によるスロットル流量
- **インマニ**: 質量収支 + 理想気体式で圧力更新
- **エンジン**:
  - 4ストローク（2回転/サイクル）
  - VEを `rpm / throttle / vvt` の関数で近似
  - 筒内取り込み空気量から燃料量・熱効率を仮定し図示トルクを算出
  - 摩擦トルク + スタータトルクと慣性で回転数更新
- **排気**: まずは定圧境界（触媒なし）
- **可視化**: ASCIIベースのRPMトレンドとp-V線図

## 注意

- 物理厳密モデルではなく、始動過渡の学習・制御検討用の「軽量0次元モデル」です。
- ターミナルを raw モードに変更して1文字入力をポーリングします。異常終了で表示が崩れた場合は `stty sane` を実行してください。
