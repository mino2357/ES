# ES

4ストロークエンジンの始動〜アイドル安定化までの過渡挙動を、Rustでリアルタイム可視化するシミュレータです。

## できること

- スターターモーターによるクランキング
- 初爆条件（スターターON + 点火ON + 燃料ON + 一定回転/吸気圧）での着火
- 着火後のトルク生成と摩擦トルクのつり合いによる回転上昇/安定化
- 吸気系（スロットル + インマニ）とエンジン吸入の連成
- 排気圧を境界条件にした簡易排気系
- VVT（吸気/排気）による充填効率変化
- GUIダッシュボード上でのリアルタイム表示
  - RPMトレンド
  - p-V 線図
  - トルク・MAP・IMEP など主要状態量
- 数値計算した排気圧（定常成分＋燃焼パルス成分）を反映した簡易排気音のリアルタイム再生

## 実行

```bash
cargo run
```

## 操作

### GUI操作

- 左パネルのスライダ・チェックボックスで各種制御を変更
- 気筒数は `2 / 3 / 4 / 6 / 8 cyl` ボタンで切り替え

### キーボードショートカット

- `q`: 終了
- `s`: スターター ON/OFF
- `i`: 点火 ON/OFF
- `f`: 燃料 ON/OFF
- `w` / `x`: スロットル開度 増/減

## モデル概要

- **吸気**: オリフィス近似によるスロットル流量
- **インマニ**: 質量収支 + 理想気体式で圧力更新
- **エンジン**:
  - 4ストローク（2回転/サイクル）
  - 気筒数をリアルタイムに切り替え可能
  - VEを `rpm / throttle / vvt` の関数で近似
  - 筒内取り込み空気量から燃料量・熱効率を仮定し図示トルクを算出
  - 燃焼位相はWiebe関数（SOC/燃焼期間/形状係数）で与え、瞬時トルクへ反映
  - 摩擦トルク + スタータトルクと慣性で回転数更新
- **排気**: 定圧境界（触媒なし）
- **可視化**: egui/eframe + egui_plot によるリアルタイム描画

## 注意

- 物理厳密モデルではなく、始動過渡の学習・制御検討用の「軽量0次元モデル」です。
